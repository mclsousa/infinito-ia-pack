# 15 Prompts para Datas Comemorativas

Use no ChatGPT, Gemini ou Claude. Preencha os campos `[entre colchetes]`.

---

## PROMPT B1 — Dia das Mães

**QUANDO USAR:** 2–3 semanas antes do segundo domingo de maio.

```
Você é copywriter de campanhas sazonais para pequenos negócios no Brasil.

Meu nicho: [NICHO]
Produto/serviço para presente: [O QUE VENDO]
Preço: [R$]
Diferencial: [POR QUE COMPRAR COMIGO]

Crie campanha completa para Dia das Mães:
1. Roteiro Reels 30 segundos (gancho + emoção + CTA)
2. Legenda para feed (máx. 200 palavras, tom carinhoso)
3. Sequência de 3 Stories
4. Mensagem para broadcast WhatsApp (máx. 5 linhas)

Tom: carinho autêntico, não comercial agressivo.
Inclua sugestão de prazo limite para encomendas.
```

---

## PROMPT B2 — Dia dos Namorados

**QUANDO USAR:** Início de junho (12 de junho).

```
Crie campanha de Dia dos Namorados para [NICHO].
Produto: [NOME] | Preço: [R$] | Público: [PERFIL]

Inclua:
- Reels 30s (gancho + oferta + CTA)
- Legenda emotiva
- 3 Stories
- Ângulo alternativo: presente para amiga, mãe, ou "presente para mim"

Tom: leve, romântico ou divertido conforme o nicho.
```

---

## PROMPT B3 — Black Friday

**QUANDO USAR:** Última semana de novembro.

```
Você é estrategista de Black Friday para MEI no Brasil.

Negócio: [NICHO]
Oferta real: [DESCONTO OU BÔNUS — sem exagerar]
Estoque ou vagas limitadas: [NÚMERO REAL]
Preço de: [R$] por: [R$]

Crie:
1. Sequência de 5 Stories (contagem + prova + CTA)
2. Roteiro Reels anúncio 45s
3. Legenda com bullets do que está incluso
4. Mensagem WhatsApp para lista de clientes

Tom: urgência honesta. Não use "maior desconto da história" se não for verdade.
```

---

## PROMPT B4 — Natal / Fim de Ano

**QUANDO USAR:** Novembro–dezembro.

```
Campanha de Natal para [NICHO].
Produto: [NOME]
Prazo final de encomendas: [DATA]
Entrega/retirada: [COMO FUNCIONA]

Roteiro Reels, legenda, ideia de embalagem ou brinde natalino, CTA para encomendas antecipadas.
Tom: acolhedor, planejamento com antecedência.
```

---

## PROMPT B5 — Volta às Aulas

**QUANDO USAR:** Janeiro e julho.

```
Conteúdo e oferta de Volta às Aulas para [NICHO].
Exemplos de nicho: papelaria, moda infantil, marmitas, cursos.

3 ideias de Reels, legenda, promoção sugerida com prazo.
Público: pais e estudantes.
CTA: link ou WhatsApp.
```

---

## PROMPT B6 — Dia Internacional da Mulher (8 de março)

**QUANDO USAR:** Primeira semana de março.

```
Campanha 8 de Março para [NICHO].
Ângulo: empoderamento + oferta (se fizer sentido para o negócio).

Reels 45s, legenda, 3 Stories.
Evite conteúdo vazio — traga valor para mulheres empreendedoras ou clientes.
Se for desconto, justifique com carinho, não exploração da data.
```

---

## PROMPT B7 — Dia do Consumidor (15 de março)

**QUANDO USAR:** 10–15 de março.

```
Oferta Dia do Consumidor para [NICHO].
Produto: [NOME] | Condição: [DESCONTO OU BRINDE]

Reels 30s, legenda, 3 frases para Stories com contagem regressiva.
CTA claro com prazo de 24–48h.
```

## PROMPT B8 — Páscoa

## PROMPT B11 — Carnaval

**QUANDO USAR:** Janeiro–fevereiro.

```
Conteúdo de Carnaval para [NICHO].
Ângulo escolhido: [promoção / humor / pausa / retomada pós-folia]

2 roteiros Reels, Stories, legenda.
Mencione se o negócio abre ou fecha no feriado — transparência gera confiança.
```

## PROMPT B12 — Ano Novo / Metas

**QUANDO USAR:** Dezembro–janeiro.

```
Conteúdo de Ano Novo para [NICHO].
Ângulo: metas, recomeço, planejamento.

Roteiro "como [SEU SERVIÇO] pode ajudar em [ANO]" ou retrospectiva do negócio.
Reels + legenda reflexiva + CTA suave.
```

## PROMPT B13 — Dia do Cliente

**QUANDO USAR:** 15 de setembro.

```
Campanha Dia do Cliente para [NICHO].
Ação: [desconto / brinde / sorteio / agradecimento sem desconto]

Reels agradecendo clientes, legenda, ideia de repost de depoimentos.
Tom: gratidão genuína.
```

## PROMPT B14 — Halloween

**QUANDO USAR:** Outubro (nichos criativos: doces, moda, festas, beleza artística).

```
Conteúdo Halloween para [NICHO].

Roteiro criativo 30–45s, legenda, ideia visual.
Tom: divertido. Se público for família com crianças, evite terror.
```

## PROMPT B15 — Aniversário da Loja

**QUANDO USAR:** Data de fundação do negócio.

```
Meu negócio completa [X] anos em [DATA].
Nicho: [NICHO]
Oferta especial: [DESCONTO / BRINDE / SORTEIO]

Roteiro storytelling da história + oferta, legenda, sequência Stories "bastidores ao longo dos anos".
Tom: celebração com o cliente, não só liquidação.
```
