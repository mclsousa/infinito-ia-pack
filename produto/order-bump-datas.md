**15 prompts extras** para criar conteúdo e ofertas em datas de alta busca no Brasil.  
Configure na Cakto como order bump no checkout do produto principal.

## PROMPT B1 — Dia das Mães

```
Você é copywriter de campanhas sazonais para pequenos negócios.

Meu nicho: [NICHO]
Produto/serviço para presente: [O QUE VENDO]
Preço: [R$]

Crie campanha completa para Dia das Mães:
1. Roteiro Reels 30s
2. Legenda emotiva (sem clichê excessivo)
3. 3 Stories
4. Texto para WhatsApp
Tom: carinho autêntico, não comercial agressivo.
```

## PROMPT B2 — Dia dos Namorados

