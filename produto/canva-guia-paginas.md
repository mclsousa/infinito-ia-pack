# Guia Página a Página — Canva

Use este guia ao montar o PDF no Canva. Copie e cole cada bloco em uma página A4 (retrato).

**Arquivo completo para colar:** gere com `scripts/gerar-pdf-completo.ps1` → `produto/CONTEUDO-INFINITO-IA-COMPLETO.md`

---

## PÁGINA 1 — CAPA

```
CONTEÚDO INFINITO IA

45 Prompts Prontos para
TikTok e Instagram

Crie posts, roteiros e legendas
no ChatGPT em 2 minutos

[Seu nome ou marca]
Versão 1.0
```

**Visual sugerido:** fundo escuro ou gradiente roxo/azul, fonte bold branca, ícone de celular ou IA discreto.

---

## PÁGINA 2 — COMO USAR

```
COMO USAR EM 3 PASSOS

1. COPIE o prompt completo (bloco de texto)

2. COLE no ChatGPT, Gemini ou Claude
   Substitua tudo que estiver [ENTRE COLCHETES]
   com dados do SEU negócio

3. PERSONALIZE o resultado antes de publicar
   Leia em voz alta — se soar robô, edite 2 frases

DICA DE OURO
Sempre adicione 2–3 frases sobre seu negócio real
no final do prompt. Isso evita texto genérico.

COMPATÍVEL COM VERSÃO GRATUITA
Não precisa pagar ChatGPT Plus.
```

---

## PÁGINAS 4–48 — PROMPTS

**Regra:** 1 prompt por página (ou 2 se prompt curto).

Para cada página use este layout:

```
┌─────────────────────────────────────┐
│ PROMPT [NÚMERO] — [NOME]            │
│ Módulo [X]                          │
├─────────────────────────────────────┤
│ QUANDO USAR: [1 linha]              │
├─────────────────────────────────────┤
│ [BLOCO DO PROMPT — fonte monospace  │
│  ou caixa cinza destacada]          │
├─────────────────────────────────────┤
│ DICA: [1 linha]                     │
└─────────────────────────────────────┘
```

Copie cada prompt de `prompts-raw.md` ou do arquivo `CONTEUDO-INFINITO-IA-COMPLETO.md`.

---

## PÁGINAS 49–50 — BÔNUS 1 (Calendário)

Copie tabelas de `bonus-calendario-30.md`.  
Pode resumir em 2 páginas (semana 1–2 na pág 49, semana 3–4 na pág 50).

---

## PÁGINA 51 — BÔNUS 2 (Guia anti-robô)

Copie conteúdo principal de `guia-uso-ia.md` (5 passos + checklist).

---

## PÁGINA 52 — ENCERRAMENTO

```
OBRIGADO!

Você agora tem 45 ferramentas para nunca mais
ficar em branco na hora de postar.

PRÓXIMOS PASSOS
□ Escolha 3 prompts e teste hoje
□ Planeje a semana com o calendário bônus
□ Poste 1 Reels usando um roteiro gerado

Dúvidas: [seu e-mail ou @instagram]

Compartilhe seus resultados — adoraria ver!

— [Sua marca]
```

---

## Exportar do Canva

1. **Baixar PDF** — qualidade impressão
2. Nome do arquivo: `Conteudo-Infinito-IA.pdf`
3. Upload no Google Drive
4. Link de entrega na Cakto

---

## Paleta de cores sugerida

| Elemento | Cor |
|----------|-----|
| Fundo capa | #1a1a2e ou #0f0f23 |
| Título | #ffffff |
| Destaque | #7c3aed ou #e94560 |
| Caixa prompt | #f4f4f5 fundo, texto #18181b |
| Fonte | Montserrat, Poppins ou similar sans-serif |

---

## Tempo estimado no Canva

| Etapa | Tempo |
|-------|-------|
| Capa + índice | 30 min |
| 45 páginas de prompts (template duplicar) | 2–3 h |
| Bônus + encerramento | 45 min |
| Revisão e export | 30 min |
| **Total** | **~4 horas** |

**Atalho:** duplique a página-template 45 vezes e só troque o texto de cada prompt.
